# Calculadora Bash

Script de calculadora desenvolvido em linguagem Shell Script (Bash),
criado como atividade prática do curso de Analista de Dados na EBAC.

## Descrição

O script realiza as quatro operações matemáticas básicas: adição,
subtração, multiplicação e divisão. Possui tratamento de erro para
divisão por zero e validação de opção inválida.

## Tecnologias utilizadas

- Shell Script (Bash)
- Utilitário bc (calculadora de precisão)

## Como executar

Clone o repositório:

   git clone https://github.com/Claudiosantosvip/calculadora-

Acesse a pasta:

    cd calculadora-bash

Dê permissão de execução ao arquivo:

    chmod u+x calculadora.sh

Execute o script:

    ./calculadora.sh

## Comandos utilizados

Criação do arquivo via editor de texto:

    nano calculadora.sh

Definição das permissões do arquivo:

    chmod 644 calculadora.sh
    chmod u+x calculadora.sh

Verificação das permissões aplicadas:

    ls -la calculadora.sh

Execução do script:

    ./calculadora.sh

## Permissões aplicadas

    -rwxr--r--

- Proprietário: leitura, escrita e execução
- Grupo: somente leitura
- Outros: somente leitura

## Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão com duas casas decimais
- Tratamento de erro para divisão por zero
- Validação de opção inválida
